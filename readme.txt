== Description ==

Ec3popupinfo is a plugin that popup message on mouse over the event-calendar plugin.
w

== Installation ==

1. Install and Settings event-calendar plugin. http://wordpress.org/extend/plugins/event-calendar/
2. Upload the entire ec3popupinfo folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Visit Settings Ec3popupinfo.
5. Input data and save.
